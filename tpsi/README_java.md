# Eseguire SempliceInterattivo

Dal folder `tpsi` compilare ed eseguire con Java (JDK installato):

```bash
cd tpsi
javac SempliceInterattivo.java
java SempliceInterattivo
```

Menu disponibile:
- `1` Echo: ripete il testo inserito
- `2` Somma: somma di due numeri con validazione
- `3` Palindromo: verifica se il testo è palindromo (caratteri alfanumerici)
- `4` Esci

È un semplice esempio didattico per input/output console in Java.
